หน่วยไตเทียม Dashboard

เชื่อมต่อ Google Apps Script เพื่อดึงข้อมูลจาก Google Sheet อัตโนมัติ

1
เปิด Google Sheet → Extensions → Apps Script
2
วางโค้ดที่ได้รับ → Deploy → New deployment → Web app
ตั้งค่า Who has access: Anyone
3
Copy Web app URL ที่ได้ วางด้านล่าง
⚠ เชื่อมต่อไม่ได้ — ตรวจสอบ URL และการ Deploy อีกครั้ง
